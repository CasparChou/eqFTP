eqFTP

FTP/SFTP client for Brackets code editor

Donate: Patreon

What is this?

eqFTP is a FTP/SFTP client for Brackets code editor.
It delivers intuitive interface to work with your remote servers, syncroniztion, remote file structure exploring and password encryption.
